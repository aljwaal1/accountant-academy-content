# تحديث محتوى أكاديمية المحاسب الدولي - 7 دروس جديدة

هذا التحديث يضيف 7 دروس جديدة:

## IFRS / IAS
- IFRS 3 - تجميع الأعمال
- IFRS 13 - قياس القيمة العادلة
- IAS 24 - الإفصاح عن الأطراف ذات العلاقة

## CMA
- الموازنات والتخطيط
- تحليل الانحرافات
- قياس الأداء وبطاقات الأداء
- الرقابة الداخلية

## طريقة الرفع
استبدل ملف `academy_content.json` الموجود في مستودع المحتوى بهذا الملف الجديد، ثم اعمل Commit.
بعد ذلك افتح التطبيق واضغط زر تحديث المحتوى.
