## طريقة الرفع الصحيحة
1. فك ضغط هذا الملف على جهازك.
2. افتح مستودع GitHub:
   aljwaal1/accountant-academy-content
3. اضغط Add file ثم Upload files.
4. اسحب الملفات الخمسة الموجودة هنا إلى المستودع:
   - academy_content.json
   - README.md
   - README_UPLOAD.txt
   - lesson_template.json
   - README_UPDATE_8_LESSONS.md
5. عند ظهور رسالة أن academy_content.json سيتم استبداله، وافق.
6. اضغط Commit changes.
7. افتح التطبيق واضغط تحديث المحتوى.
