# محتوى أكاديمية المحاسب الدولي

هذا المستودع مخصص لتحديث محتوى تطبيق أكاديمية المحاسب الدولي بدون إصدار APK جديد.

## طريقة الاستخدام

1. ارفع ملف `academy_content.json` إلى مستودع GitHub.
2. افتح الملف من GitHub.
3. اضغط زر Raw.
4. انسخ رابط Raw وضعه في التطبيق أو ثبته داخل التطبيق.
5. عند تعديل المحتوى، اعمل Commit فقط.
6. داخل التطبيق اضغط: التحقق من التحديثات الآن.

## ملاحظات مهمة

- لا تغيّر أسماء الحقول الأساسية مثل: `tracks`, `lessons`, `questions`.
- عند إضافة درس جديد، اجعل `id` فريدًا ولا تكرره.
- رقم الإجابة الصحيح في `answer` يبدأ من صفر.
- 0 يعني الخيار الأول، 1 يعني الثاني، 2 يعني الثالث، 3 يعني الرابع.
- المحتوى تعليمي مبسط وليس نصًا رسميًا للمعايير.
- لا تنسخ نصوص المعايير الرسمية حرفيًا.
